# ЛР1 — Figma-спецификация под Ant Design

## Цель
Подготовить UI/UX “контракт” для разработки (экраны, флоу, компоненты, состояния).

## Сделать
- Figma страницы: Flows, UI, Components, States
- 2 сценария:
  - Student: Login → My Tickets → Create → Details → Comment
  - Operator: Queue(New) → Details → Assign → Change status / Reject
- Экраны (min 6, лучше 8):
  - Login, Register, Tickets List, Create Ticket, Ticket Details, Queue New, Assigned, Admin Categories (опц. Users)
- Components (AntD-похожие): Layout/Menu/Table/Form/Input/Select/Tag/Modal/Alert/Spin/Result
- States:
  - List: loading/empty/error
  - Create: validation/submitting/success/error
  - Details: loading/not found/error
- 2 модалки: ConfirmDialog + RejectTicketDialog

## Сдать
Ссылка на Figma + Prototype + список сценариев.
