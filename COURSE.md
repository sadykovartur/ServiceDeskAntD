# План курса: 8 лабораторных по 3 часа

Сквозной проект: **Service Desk** (тикеты, комментарии, роли, статусы).

- ЛР1 — Figma-спецификация под Ant Design
- ЛР2 — Каркас клиента (AntD + Router), без API + ESLint/Prettier + launch.json
- ЛР3 — Backend основа: EF Core + PostgreSQL + миграции + Swagger
- ЛР4 — Интеграция клиента с API (Tickets list + details)
- ЛР5 — Create Ticket “end-to-end” (AntD Form + валидация)
- ЛР6 — Identity + JWT + роли + AuthContext
- ЛР7 — Workflow оператора + комментарии + модалки
- ЛР8 — Admin + качество + CI/Docker

Оценивание (пример): каждая ЛР = 10 баллов (6 обяз., 2 качество, 2 бонус).
